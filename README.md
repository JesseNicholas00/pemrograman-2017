# Pemrograman GEMASTIK 2017

Arsip pemrograman **GEMASTIK 2017** (**GEMASTIK 10**), berisi:

- soal
- pembangkit kasus uji
- solusi
- pembahasan

| A     | `pantun`      | Berbalas Pantun       | Alham Fikri Aji   |

| C     | `sc`          | Penulis Soal          | Ashar Fuadi       |
